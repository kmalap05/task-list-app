# Task List App

A simple task list web app where users can add, delete, and mark tasks as completed.

## 🚀 Features

- ✅ Add new tasks  
- ❌ Delete tasks  
- 🎯 Mark tasks as completed  
- 🔄 Persist tasks using local storage

## 🛠️ Technologies Used

- HTML  
- CSS  
- JavaScript
